I challenged myself to simulate
and detect ONE new cyber attack
every single day for 30 days!

Each day I:
- Perform attack from Kali Linux
- Detect using raw logs FIRST
- Confirm in Splunk and Wireshark
- Write professional incident report
- Document everything here
